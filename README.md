## Steps

```
1. Create BrigingSwiftAndObjC project using objective c as the programming language.
```

```
2. Add new swift file . It will ask to create briding header. Select Create Bridging Header option.
```

```
3. Import "BrigingSwiftAndObjC-Swift.h" into ViewController.m
```

```
4. Make sue to import all the new objective c files except ViewController.h into BrigingSwiftAndObjC-Bridging-Header.h
```


```
Check the header file by clicking at the project root directory, selecting target suboption and fill BrigingSwiftAndObjC-Bridging-Header at swift compiler options general Bridging Header File
```
